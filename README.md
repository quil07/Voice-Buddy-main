\#  VoiceBuddy



VoiceBuddy is a desktop voice assistant built with Python that supports

voice and text commands to open apps, websites, and perform searches.



\##  Features

\- Voice + text command support

\- Open installed Windows apps automatically

\- Open websites and URLs

\- Google search fallback

\- Fuzzy matching for natural commands

\- Offline system commands (calculator, notepad, cmd)

\- Modern UI using CustomTkinter



\##  Tech Stack

\- Python

\- CustomTkinter

\- SpeechRecognition

\- pyttsx3

\- Windows Start Menu indexing



\##  How to Run



```bash

pip install customtkinter pyttsx3 SpeechRecognition

pip install pipwin

pipwin install pyaudio

python app.py
